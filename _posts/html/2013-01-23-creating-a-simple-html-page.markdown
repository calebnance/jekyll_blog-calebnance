The very basics of creating an HTML page. Hello World  

Create a file and call it `"index.html"` and place the following into it:  
{% highlight html %}
<html>
  <head>
    <title>Hello World</title>
  </head>
  <body>
    <h1>Regular H1 Tag</h1>
  </body>
</html>
{% endhighlight %}
